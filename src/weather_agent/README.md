# 项目说明

## 简介
本项目由初始化脚本自动生成，包含基本目录结构和虚拟环境。

## 使用方法
1. 激活虚拟环境： .venv\Scripts\activate
2. 安装依赖： pip install -r requirements.txt
3. 在 src/ 目录编写代码，在 tests/ 目录编写测试

## 目录结构
- src/      → 源代码（已生成 __init__.py）
- tests/    → 单元测试（已生成 __init__.py）
- docs/     → 文档
- data/     → 数据文件

## 注意事项
- 请勿提交 .venv/ 和 .env 文件到 Git 仓库
- 请在 requirements.txt 中维护依赖清单
